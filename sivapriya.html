🛠️ How to enable email alerts when a patient books
1
Go to Supabase Dashboard → Database → Extensions and enable pg_net and pgcrypto
2
Go to Database → Functions, create a function that calls an external email API (e.g. Resend.com or SendGrid) via pg_net.http_post() with the appointment details.
3
Go to Database → Triggers, create an AFTER INSERT trigger on the appointments table that calls your function — it will fire every time a patient books.
4
Alternative (easier): Use Supabase Webhooks (Database → Webhooks) to POST to a free service like n8n.io or make.com which then sends an email via Gmail — no code required!
